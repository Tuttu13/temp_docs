# ユーザーマニュアル（利用者向け）

## 📌 概要
画面説明・操作手順を記載した一般ユーザー向けマニュアルです。

## 🚀 はじめに

### アカウント作成

1. トップページの「新規登録」をクリック
2. 必要情報を入力:
   - メールアドレス
   - ユーザー名
   - パスワード（8文字以上）
3. 利用規約を確認し、チェックボックスにチェック
4. 「登録」ボタンをクリック
5. 登録したメールアドレスに確認メールが届く
6. メール内のリンクをクリックして認証完了

### ログイン

1. トップページの「ログイン」をクリック
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック
4. ダッシュボードが表示される

### パスワードを忘れた場合

1. ログインページの「パスワードを忘れた方はこちら」をクリック
2. 登録したメールアドレスを入力
3. 「送信」をクリック
4. メールに届いたリンクをクリック
5. 新しいパスワードを設定

## 👤 プロフィール設定

### プロフィールの編集

1. 画面右上のユーザーアイコンをクリック
2. 「プロフィール設定」を選択
3. 編集したい項目を変更:
   - ユーザー名
   - 自己紹介
   - アバター画像
   - Webサイト
4. 「保存」をクリック

### プロフィール画像の変更

1. プロフィール設定画面を開く
2. アバター画像エリアをクリック
3. 画像ファイルを選択（JPG, PNG, 最大5MB）
4. 画像の位置を調整
5. 「保存」をクリック

### パスワード変更

1. プロフィール設定画面を開く
2. 「セキュリティ」タブをクリック
3. 「パスワード変更」セクションで:
   - 現在のパスワード
   - 新しいパスワード
   - 新しいパスワード（確認用）
4. 「変更」をクリック

## ✍️ 投稿機能

### 新規投稿の作成

1. ダッシュボードの「投稿作成」ボタンをクリック
2. 投稿内容を入力:
   - タイトル（必須）
   - 本文（必須）
   - カテゴリ（必須）
   - タグ（任意）
3. プレビューで確認
4. 「下書き保存」または「投稿」をクリック

**タイトルのヒント**:
- 明確で分かりやすく
- 3〜60文字を推奨
- キーワードを含める

**本文のヒント**:
- 段落を分けて読みやすく
- 見出しを活用
- 画像・動画で補足

### 投稿の編集

1. マイページ > 投稿一覧
2. 編集したい投稿の「編集」ボタンをクリック
3. 内容を修正
4. 「更新」をクリック

### 投稿の削除

1. マイページ > 投稿一覧
2. 削除したい投稿の「削除」ボタンをクリック
3. 確認ダイアログで「削除」をクリック

**注意**: 削除した投稿は復元できません

### 画像の挿入

1. 投稿作成/編集画面で画像アイコンをクリック
2. 画像ファイルを選択またはドラッグ&ドロップ
3. 画像の説明（alt属性）を入力
4. 「挿入」をクリック

**対応形式**: JPG, PNG, GIF, WebP  
**最大サイズ**: 10MB

## 💬 コメント機能

### コメントの投稿

1. 投稿詳細ページを開く
2. ページ下部のコメント欄に入力
3. 「コメント」ボタンをクリック

### コメントの編集

1. 自分のコメントの「編集」リンクをクリック
2. 内容を修正
3. 「更新」をクリック

### コメントの削除

1. 自分のコメントの「削除」リンクをクリック
2. 確認ダイアログで「削除」をクリック

## 🔍 検索機能

### 投稿を検索

1. ヘッダーの検索ボックスにキーワードを入力
2. Enterキーを押すまたは検索アイコンをクリック
3. 検索結果が表示される

### フィルター機能

検索結果画面で以下でフィルター可能:
- カテゴリ
- タグ
- 投稿日
- 著者

### 並び替え

検索結果を以下で並び替え可能:
- 関連度順（デフォルト）
- 新着順
- 人気順
- 古い順

## ⭐ お気に入り・ブックマーク

### お気に入りに追加

1. 投稿詳細ページを開く
2. ハートアイコンをクリック
3. お気に入りに追加される

### お気に入りの確認

1. マイページ > お気に入り
2. お気に入り登録した投稿一覧が表示される

### お気に入りから削除

1. お気に入り一覧または投稿詳細ページ
2. ハートアイコンをもう一度クリック

## 🔔 通知機能

### 通知の種類

- コメントへの返信
- お気に入り登録された
- フォローされた
- メンション（@username）

### 通知の確認

1. ヘッダーのベルアイコンをクリック
2. 通知一覧が表示される
3. 通知をクリックして該当ページへ移動

### 通知設定

1. プロフィール設定 > 通知設定
2. 受け取りたい通知をオン/オフ
3. 「保存」をクリック

## 🎨 テーマ設定

### ダークモード切り替え

1. ヘッダーのテーマアイコンをクリック
2. ライト/ダーク/システム設定を選択

## ⚙️ アカウント設定

### メール通知設定

1. プロフィール設定 > メール設定
2. 受け取りたいメール通知を選択:
   - 新規コメント
   - お気に入り
   - フォロー
   - 週次ダイジェスト
3. 「保存」をクリック

### プライバシー設定

1. プロフィール設定 > プライバシー
2. 公開範囲を設定:
   - プロフィールの公開範囲
   - 投稿の公開範囲
   - コメントの許可
3. 「保存」をクリック

### アカウントの削除

1. プロフィール設定 > アカウント
2. 「アカウント削除」をクリック
3. パスワードを入力して確認
4. 「削除」をクリック

**注意**: アカウント削除は取り消せません。すべてのデータが完全に削除されます。

## ❓ よくある質問（FAQ）

### アカウント関連

**Q: ユーザー名は変更できますか？**  
A: はい、プロフィール設定からいつでも変更できます。

**Q: メールアドレスは変更できますか？**  
A: はい、プロフィール設定 > アカウント から変更できます。確認メールが送信されます。

### 投稿関連

**Q: 投稿の文字数制限はありますか？**  
A: タイトルは255文字、本文は50,000文字までです。

**Q: 下書きはどこに保存されますか？**  
A: マイページ > 下書き一覧 で確認できます。

**Q: 投稿を予約投稿できますか？**  
A: はい、投稿作成画面で「予約投稿」を選択し、日時を設定できます。

### その他

**Q: 問題が発生したら？**  
A: お問い合わせフォームまたはsupport@example.comまでご連絡ください。

**Q: フィードバックを送りたい**  
A: フィードバックフォームから送信できます。いつでも歓迎します！
