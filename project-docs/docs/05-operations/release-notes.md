# リリースノート

## 📌 概要
変更履歴とバージョン情報を記録するドキュメントです。

## 🚀 v1.0.0 - 2025-11-15

### 🎉 初回リリース

プロダクションリリース第1版

### ✨ 新機能

- **ユーザー認証**: JWT による認証機能
- **投稿管理**: CRUD操作、マークダウンサポート
- **コメント機能**: ネストコメント対応
- **検索機能**: 全文検索、フィルター、ソート
- **お気に入り**: 投稿のブックマーク機能
- **通知システム**: リアルタイム通知
- **管理画面**: ユーザー・コンテンツ管理

### 🔒 セキュリティ

- パスワードのbcryptハッシュ化
- CSRF保護
- XSS対策
- SQLインジェクション対策
- レート制限

### 📱 対応環境

- **ブラウザ**: Chrome 100+, Firefox 100+, Safari 15+, Edge 100+
- **モバイル**: iOS 15+, Android 10+
- **画面解像度**: 320px〜4K対応

## 🔄 アップグレードガイド

### v0.9.0 から v1.0.0 へ

#### データベースマイグレーション

```bash
# バックアップ
npm run db:backup

# マイグレーション実行
npm run migration:run

# 確認
npm run migration:show
```

#### 環境変数の変更

```bash
# 新規追加
FEATURE_DARK_MODE=true
EMAIL_NOTIFICATIONS=true

# 変更
# JWT_EXPIRES_IN=1h → JWT_EXPIRES_IN=15m
```

#### 破壊的変更

**API変更**:
- `GET /api/users` のレスポンス構造変更
  ```json
  // Before
  { "users": [...] }
  
  // After
  { 
    "data": [...],
    "pagination": {...}
  }
  ```

**コンポーネント名変更**:
- `UserList` → `UserTable`
- `PostCard` → `PostPreview`
