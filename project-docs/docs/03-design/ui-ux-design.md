# UI/UX設計書

## 📌 概要
ワイヤーフレーム、デザインガイドラインを定義するドキュメントです。

## 🎨 デザインガイドライン

### カラーパレット

| 用途 | カラー名 | カラーコード | 使用例 |
|------|---------|------------|--------|
| プライマリ | Blue | `#0066CC` | ボタン、リンク |
| セカンダリ | Gray | `#6C757D` | 補助テキスト |
| 成功 | Green | `#28A745` | 成功メッセージ |
| 警告 | Yellow | `#FFC107` | 警告メッセージ |
| エラー | Red | `#DC3545` | エラーメッセージ |
| 背景 | White | `#FFFFFF` | ページ背景 |
| テキスト | Dark Gray | `#212529` | 本文テキスト |

### タイポグラフィ

#### フォントファミリー
- **見出し**: 'Noto Sans JP', sans-serif
- **本文**: 'Noto Sans JP', sans-serif
- **コード**: 'Fira Code', monospace

#### フォントサイズ

| 要素 | サイズ | 行高 | 用途 |
|------|--------|------|------|
| H1 | 32px | 1.5 | ページタイトル |
| H2 | 24px | 1.5 | セクション見出し |
| H3 | 20px | 1.5 | サブセクション |
| Body | 16px | 1.6 | 本文 |
| Small | 14px | 1.4 | 補足テキスト |
| Caption | 12px | 1.3 | キャプション |

### スペーシング

| サイズ | 値 | 用途 |
|--------|-----|------|
| xs | 4px | 微小な余白 |
| sm | 8px | 小さな余白 |
| md | 16px | 標準余白 |
| lg | 24px | 大きな余白 |
| xl | 32px | 特大余白 |

### コンポーネント

#### ボタン

**プライマリボタン**:
- 背景色: `#0066CC`
- テキスト色: `#FFFFFF`
- パディング: `12px 24px`
- ボーダー半径: `4px`

**セカンダリボタン**:
- 背景色: 透明
- ボーダー: `1px solid #0066CC`
- テキスト色: `#0066CC`

#### 入力フィールド

- 高さ: `40px`
- ボーダー: `1px solid #CED4DA`
- ボーダー半径: `4px`
- フォーカス時: ボーダー色 `#0066CC`

## 📱 レスポンシブデザイン

### ブレークポイント

| デバイス | 幅 | 備考 |
|---------|-----|------|
| Mobile | < 768px | スマートフォン |
| Tablet | 768px - 1024px | タブレット |
| Desktop | > 1024px | PC |

### レスポンシブ対応

- **モバイル**: 1カラムレイアウト
- **タブレット**: 2カラムレイアウト
- **デスクトップ**: 3カラムレイアウト

## 🎭 インタラクション

### アニメーション

| 要素 | アニメーション | 時間 | イージング |
|------|--------------|------|-----------|
| ボタンホバー | 背景色変化 | 0.2s | ease-in-out |
| モーダル表示 | フェードイン | 0.3s | ease-out |
